
# Ejercicios de JavaScript – FreeCodeCamp

Este repositorio contiene mis ejercicios y prácticas de JavaScript realizados mientras estudio el curso de FreeCodeCamp.

Aquí vas a encontrar ejercicios básicos y avanzados que me ayudan a entender la lógica, estructuras de control, funciones, arrays y más.

## Contenido

- Variables y tipos de datos  
- Condicionales y operadores  
- Bucles (`for`)  
- Funciones  
- Arrays y métodos  
- Objetos básicos  

## Cómo usarlo

Cada archivo `.js` corresponde a un ejercicio o práctica. Podes abrirlos con cualquier editor de texto o correrlos en Node.js para probarlos.
